# address_book

#### 介绍
address_book version 1.0.1

#### 软件架构
Qt + 数据库sqlite


#### 安装教程

1.  安装QT5
2.  安装数据库sqlite

#### 使用说明

1.  该版本支持添加、删除、双击查看详细信息、查询、刷新等功能
2.  1.0.2 版本将推出“发送信息”功能
